# WeChat Tech Article Templates (Copy & Paste)

本文件只放“可复制模板”，避免把 `skills/wechat-tech-article/SKILL.md` 撑得太长。

风格与硬规则以 `skills/wechat-tech-article/references/style-guide.md` 为准；这里提供的是“开写最快路径”。

---

## 模板 1：开源工具/项目发布（模式 A）

````md
# [一句话钩子标题：反问/反差/收益点/“我开源了一个 X”】【标题后 10-15 行内给入口链接 + **tagline**】

![Image]([首图路径或链接])

**[一句话定位 tagline：这是什么 + 适合谁 + 解决什么]**

项目地址：[GitHub/下载/在线体验链接]

---

# 缘起：我为什么要做/要换

- [真实痛点 1]
- [真实痛点 2]
- [真实痛点 3]

---

# 核心能力（收益 + 触发方式）

- [能力 1：收益 + 命令/快捷键/按钮]
- [能力 2：收益 + 命令/快捷键/按钮]
- [能力 3：收益 + 命令/快捷键/按钮]

---

# 如何使用（可复制为王）

```bash
[最短安装路径]
```

```bash
[最短使用示例（一个真实例子）]
```

---

# 设计选择 / 为什么这么做（可选但很加分）

- [选择 1：原因 + 好处]
- [选择 2：原因 + 好处]

---

# 适用 / 不适用结论（一定要写）

- 如果你……，建议用……
- 如果你……，更适合……

---

# 写在最后

- 后续规划：……
- 欢迎反馈/Star/试用：……
````
